关于 / 帮助

本页面是一个 Minecraft 每日信息聚合站, 每天自动从官方数据源抓取 Java 版 / 基岩版 / 网易版的最新版本、特性、新闻等数据, 展示在 Actions & Stuff 材质包风格的按钮 + Minecraft Wiki 风格的布局中.

数据源

  • Java 版: Mojang version_manifest.json (官方 API, 包含所有历史版本)
  • 基岩版: feedback.minecraft.net 官方更新说明 + 公开版本号
  • 网易版: mc.163.com 网易官方 + 公开资料

自动刷新

通过 scraper/fetch_minecraft_news.py + Windows 任务计划程序每日执行 + Github Actions, 把结果写入 data/latest.json, 页面打开/手动刷新时读取.

项目结构

minecraft-news/
├─ index.html
├─ css/style.css
├─ js/app.js
├─ data/
│  ├─ latest.json
│  ├─ java.json
│  ├─ bedrock.json
│  ├─ netease.json
│  └─ history/  (按日期归档)
├─ scraper/fetch_minecraft_news.py
└─ logs/  (按日期记录)
